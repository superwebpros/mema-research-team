## DESIGN + BUILD — Block 2 (Slides 11-20) | 9:30-10:30

*"Decide who does what. Deploy it into the workflow that already exists."*

### Slide 11: Design — "Decide Who Does What"
- Compound Stage 3 framing
- "The output here is clarity: which tasks stay with your people, which move to AI, and who owns every handoff."
- For the market report: what does the human do vs. what does the skill do?
  - Human: curates sources, provides judgment, reviews accuracy
  - Skill: structures output, matches format, maintains consistency
  - Handoff: human provides raw inputs → skill produces draft → human reviews

### Slide 12: What Is a Skill?
- Definition: reusable workflow = instructions + examples + steps
- "A recipe for AI"
- Both ChatGPT and Claude have built-in skill wizards — you don't write from scratch
- This is an open standard (agentskills.io) — transferable across tools
- Visual: Instructions → Examples → Steps → Consistent Output

### Slide 13: Anatomy of a Skill
- Instructions = system prompt (what it knows, how it behaves)
- Examples = few-shot learning (what good output looks like)
- Steps = agent workflow (runs the same way every time)
- Connect to periodic table: Skill = Pr + Ag packaged together

### Slide 14: Key Authoring Tips
- Be concise — the AI is already smart
- Show don't tell — one good example > a page of rules
- Match specificity to risk — strict for format, flexible for creative judgment
- Add accuracy guardrails — "Never fabricate statistics"
- Test and iterate — each round gets better

### Slide 18: Sharing Skills + Data/Privacy
- How to share (profile → Skills → ... → Share)
- "One person builds it, the whole team uses it"
- Where your data lives:
  - Skills → ChatGPT workspace (Enterprise DPA)
  - Conversations → not used for training
  - Uploaded files → indexed by OpenAI

### Slide 19: Connectors — Linking Your Data Sources
- How connectors work: Sync → Chunk (~800 tokens) → Embed → Semantic search
- Limitations:
  - Fixed chunking breaks tables/structured content
  - No metadata filtering (doesn't use SharePoint taxonomies)
  - Periodic sync (hours, not real-time)
- Context windows: ChatGPT 128K vs. Claude 200K

### Slide 20: The Fix — Document Discipline
- Metadata header block at top of every document (show example)
- Consistent file naming convention
- "No pipeline needed. Just document discipline."
- "This works with every AI tool because they all index text."
